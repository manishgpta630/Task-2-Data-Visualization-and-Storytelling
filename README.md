# Task-2-Data-Visualization-and-Storytelling

## Objective
Create an interactive sales dashboard using Power BI to analyze the Superstore dataset and present meaningful business insights.

## Tools Used
- Power BI Desktop
- Superstore.csv

## Dashboard Features
- KPI Cards (Total Sales, Total Profit, Total Quantity)
- Sales by Category
- Profit by Category
- Sales by Region
- Monthly Sales Trend
- Interactive Slicers (Region, Category, Year)

## Business Insights
- Technology category generated the highest sales.
- West region recorded the highest sales.
- Sales increased during the last quarter of the year.
- Furniture generated lower profit compared to Technology and Office Supplies.
